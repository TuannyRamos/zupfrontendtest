Teste para vaga de Front-End Developer
==================

# Dependências

Para rodar este projeto você vai precisar de
- Node JS
- Gulp CLI

Após isto basta seguir os passos abaixo:

# Instalação

```
npm install 
```

# Para executar o projeto

```
npm start
```

# Utilizar o projeto

Basta abrir o navegador e acessar o seguinte endereço:
```
http://localhost:8080/
```
